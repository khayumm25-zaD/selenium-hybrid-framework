# Selenium Hybrid Automation Framework

## Overview
This is a Hybrid Selenium Automation Framework developed using Java, Selenium WebDriver, TestNG, Maven, and Page Object Model (POM) design pattern.

The framework is designed for scalable, reusable, and maintainable test automation.

---

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Extent Reports
- WebDriverManager

---

## Framework Features

- Page Object Model (POM)
- Reusable Utilities
- Explicit Wait Handling
- TestNG Listeners
- Extent HTML Reports
- Screenshot Capture on Failure
- Assertion Handling
- Hybrid Framework Architecture
